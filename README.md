## About This Portfolio

This isn't your average portfolio. **ERROR 404** is a custom-built, static portfolio designed to be at the absolute peak of creativity and animation. No Bootstrap templates were harmed in the making of this — it's 100% handcrafted.

### Features

| Feature | Description |
|---|---|
| **Particle Canvas** | Interactive particle system with mouse repulsion and dynamic connection lines |
| **Custom Cursor** | Dot + ring cursor that morphs on hover over interactive elements |
| **Magnetic Buttons** | Elements subtly follow your mouse for a tactile feel |
| **3D Tilt Cards** | Projects, skills, and testimonials respond to mouse position in 3D |
| **Floating Gradient Orbs** | Large, blurred, mouse-tracking color blobs for depth |
| **Rotating Hero Text** | Auto-cycling words: Experiences → Interfaces → Solutions → Magic |
| **GSAP Scroll Animations** | Text reveals, fade-ups, counter animations, skill bar fills |
| **Animated Timeline** | Progress line fills as you scroll through experience |
| **Parallax Big Text** | Horizontal scrolling marquee driven by scroll position |
| **Animated Loader** | Letter-by-letter reveal with pulsing blocks and progress bar |
| **Project Filtering** | Smooth filter transitions for Web / App / Design categories |
| **Noise Overlay** | Subtle grain texture for that premium feel |
| **Fully Responsive** | Looks stunning on every screen size |

### Sections

```
Home → Marquee Strip → About → Skills → Projects → Big Text → Experience → Testimonials → Contact → Footer
```

## Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, gradients, animations, grid, flexbox
- **JavaScript (Vanilla)** — No jQuery, no bloat
- **GSAP 3** — ScrollTrigger, ScrollToPlugin for scroll-driven animations
- **Bootstrap Icons** — Icon library via CDN
- **Google Fonts** — Syne, Inter, Space Grotesk

> Zero build tools. Zero dependencies to install. Just open `index.html` and go.

## Customization

All placeholder content is ready to be swapped out:

- **Name & title** — Edit the hero section in `index.html`
- **About text & photo** — Update the about section + drop your image in `assets/img/`
- **Skills** — Modify skill tags and categories to match your stack
- **Projects** — Replace placeholder cards with your real work
- **Experience** — Update the timeline with your journey
- **Contact info** — Add your real email, phone, and socials
